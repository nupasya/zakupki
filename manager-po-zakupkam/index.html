Логика будущей LMS для этого раздела:
  1. Сотрудник заходит в раздел Менеджер по закупкам.
  2. Сначала проходит Часть 1.
  3. После завершения части 1 автоматически открывается Часть 2.
  4. По каждой части можно считать прогресс, время прохождения и открывать доступ по отделам.
